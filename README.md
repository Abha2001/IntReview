# IntReview
An app that records various interview responses and publicise them.

## Build Setup
#### Dependencies

Python 3.6.3
<br>
Django 2.1.1


```bash
#clone the directory
git clone https://github.com/arc9693/IntReview.git

#change directory
cd IntReview

#Do migrations
python manage.py makemigrations
python manage.py migrate

#Run server
python manage.py runserver
```
